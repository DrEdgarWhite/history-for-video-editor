총 편집비 계산 방식

편집 비용은 시간보다, 시청 유지와 몰입 설계에 따라 달라집니다.

공식

총 편집비 = 1차 컷편집비 + 선택한 연출비

1차 컷편집은 원본 기준, 2차 연출은 최종 편집본 기준으로 계산됩니다.

계산 예시

원본 180분을 10분으로 줄이고, 기본 전달형을 적용하는 경우

  • 1차 컷편집비: 180분 × 200원 = 36,000원
  • 연출비: 10분 × 20,000원 = 200,000원
  • 총 편집비: 236,000원

수정 기준

수정은 기본 작업 범위 안에서 가능한 수정과, 새로 만드는 작업에 가까운 추가 작업으로 나뉩니다.

기본 수정

이미 만든 결과물을 다듬는 수정입니다.

  • 컷 길이 조정
  • 자막 오타 수정
  • 색보정 조정
  • 기존 효과 제거 / 수정
  • BGM 볼륨 조정

추가 작업

편집 흐름을 다시 만들거나 작업량이 늘어나는 경우입니다.

  • 인서트 / 새 장면 대량 추가
  • 외부 자료 서칭
  • 연출 재구성
  • BGM 교체 후 재구성
  • 자막 스타일 재설계

사전 기준 필요

기준이 없으면 반복 수정이 발생하기 쉬운 항목입니다.

  • 인서트 사용 구간
  • 자막 크기 / 색상 / 위치
  • 외부 이미지 사용 기준
  • 인서트 초안 방식
  • 참고 영상 기준

기본 수정은 작업 범위 안에서 반영 가능하지만,
전체 흐름을 다시 구성하거나 새로운 자료·연출을 추가하는 요청은 추가 작업으로 분류될 수 있습니다.
